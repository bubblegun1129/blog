让人上瘾的 Hexo

昨天跟工程师同事聊到一直想要使用 bootstrap 或者 Wordpress 搭建个人网站，无奈没有编程基础，进度非常十分缓慢。同事推荐了Hexo ，浏览了中文版的操作指南以后大概了解到这是一个非常智能小巧可玩性又高的博客框架，完全能够满足我的需求。
发布博文只需要在资源文件下新增md文件，使用 Markdown 简单语法就能完成排版，这一点让我毫不犹豫的放弃之前的博客项目，决定用 hexo 做一个全新的博客。
经历过近10个小时的摸索，新版的博客终于成功上线，我也能够写下这篇文章，记录一下「小白视角」博客实现流程，分享给没有编程基础却想要搭建自己个人网站的朋友。

安装 Hexo 完成建站

我参考 Hexo中文操作指南 的步骤一步步配置电脑环境，依次下载了Xcode/Node.js/git之后，运行电脑终端（我使用的 Mac 系统，所以这个输入命令行的东西叫做 终端 ）输入命令行完成 Hexo 安装。后续的建站和简单的配置都可以参考文档完成（此处不详细记录主要是因为在初期大多数配置参数无需修改）。

完成站点的维护迭代绑定域名最终发布上线我查阅了另外更加详细的资料，分别是：

  • 绑定域名 这篇文章非常详细，前半篇是对 Hexo 安装配置的解释，后半篇内容为阿里云域名的绑定解析，按照步骤操作完成后，博客就真正上线了。

  • 想要让网站页面样式美观而暂时没有能力或精力自定义页面样式时，选择一套适合喜欢的主题模版是更加方便的选择，我参考博客配置与主题修改 完成了博客主题样式的修改。

  • 学习使用 Markdown 语法写作我查看了简书官方的 Markdown新手指南
